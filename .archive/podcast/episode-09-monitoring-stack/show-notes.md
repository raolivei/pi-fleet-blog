# Episode 9: The Monitoring Stack - Show Notes

**Prometheus, Grafana, observability**

## Episode Summary

What good is a cluster if you can't see what's happening inside?
We deploy Prometheus, Grafana, and build dashboards that actually
help us understand our infrastructure.
